# Keshav Kalani | Full-Stack Portfolio

A high-performance, interactive personal portfolio built with React and Vite. This project features advanced WebGL backgrounds, custom animations, and a seamless Dark/Light theme system.

## ✨ Features

- **Dynamic Hero Section:** Featuring decrypted text animations and interactive particle systems.
- **Theme Engine:** Fully integrated Dark and Light modes with persistent local storage.
- **Interactive Backgrounds:**
  - **PixelBlast:** A 3D interactive grid on the About page.
  - **ShapeGrid:** A clean, diagonal-moving grid for the Projects section.
  - **RippleDots:** Physics-based interactive dots for the Contact section.
  - **LetterGlitch:** Matrix-style scrambled text background for the Skills page.
- **GSAP Animations:** High-end character shuffling in the footer using GSAP and ScrollTrigger.
- **Single Page Architecture:** Smooth scroll-based navigation with intersection observer highlighting.
- **Automated Contact Form:** Directly integrated with Google Forms for easy message management.

## 🛠️ Tech Stack

- **Framework:** React 19 (Vite)
- **Styling:** Vanilla CSS (Custom Variable System)
- **Animations:** GSAP, Framer Motion, ReactBits
- **3D/Graphics:** Three.js, OGL
- **Deployment:** Vercel (Recommended)

## 🚀 Getting Started

1. **Clone the repository**
2. **Install dependencies:**
   ```bash
   npm install --legacy-peer-deps
   ```
3. **Run in development mode:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```
